3. Build
--------

Dependency: 

cmake > 2.8<br />
boost > 1.55

Build:

cd FudanDNN<br />
mkdir build<br />
cd build

--use makefile<br />
cmake ../<br />
make

--use other available IDE, such as Xcode<br />
cmake -G Xcode ../

use cmake -h to get information about your IDE
